# Features Boundary

This folder is reserved for domain-driven frontend features.

Planned feature modules:

- `auth`
- `users`
- `patients`
- `practitioners`
- `sessions`
- `payments`
- `articles`
- `chat`
- `reviews`
- `settings`

Business logic should be added here gradually while `src/components` stays focused
on reusable view primitives and section-level UI.
