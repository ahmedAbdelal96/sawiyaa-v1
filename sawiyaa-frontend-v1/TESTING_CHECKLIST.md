## 3. Patient Area

### `/patient`

- [ ] الـ navigation ظاهر وشغال

- [ ] حقل التاريخ شغال صح

### `/patient/practitioners`

- [ ] نفس تجربة صفحة المختصين العامة بس جوه الـ patient shell

### Booking Flow (من صفحة المختص العامة)

**السيناريو 1: غير مسجل**
- [ ] الضغط على slot يفتح شاشة التأكيد
- [ ] بيظهر نص "سجّل دخولك لحجز هذا الموعد"
- [ ] زرار "تسجيل الدخول للحجز" يوديك `/signin/patient`

**السيناريو 2: مسجل كمختص أو أدمن**
- [ ] بعد الضغط على slot تظهر رسالة "الحجز لحسابات المرضى فقط"
- [ ] مفيش زرار حجز يتعرض

**السيناريو 3: مسجل كمريض**
- [ ] الضغط على slot يفتح شاشة التأكيد
- [ ] الموعد المختار بيتعرض بتفاصيل كاملة (يوم + وقت)
- [ ] اختيار المدة (30 أو 60 دقيقة) شغال — الـ active state بيتغير
- [ ] زرار "العودة للمواعيد" بيرجع لقائمة الـ slots
- [ ] زرار "تأكيد الحجز" بيعمل `POST /patients/me/sessions`
- [ ] زرار في حالة loading (معطل + نص "جارٍ الحجز…")
- [ ] **شاشة النجاح:**
  - [ ] بتظهر بعد الحجز الناجح
  - [ ] تاريخ ووقت الجلسة بيتعرض
  - [ ] **تحذير واضح: "في انتظار الدفع"** (مش "تم الحجز نهائيًا")
  - [ ] نص "جلستك محجوزة لمدة 15 دقيقة" بيتعرض
  - [ ] زرار "انتقل لجلساتي" يوديك `/patient/sessions`
  - [ ] زرار "حجز موعد آخر" يرجع لقائمة الـ slots
- [ ] **حالة 409 (تعارض):** رسالة "هذا الموعد لم يعد متاحًا"
- [ ] **حالة خطأ عام:** رسالة "حدث خطأ ما، يرجى المحاولة"
