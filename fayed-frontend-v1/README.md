# Sawiyaa Frontend Base

Clean admin frontend baseline for the Sawiyaa web project.

## Included

- App layout shell
- Sidebar and header
- Auth pages
- Theme switching
- i18n setup
- Placeholder routes for core product areas

## Current navigation

- Dashboard
- Patients
- Practitioners
- Sessions
- Payments
- Articles
- Chat
- Reviews
- Training
- Settings
- Admin / Operations

## Purpose

This project is intentionally cleaned from template demo content so it can be used as a base frontend for real product development.
