# Learning Products IA Note

Date: 2026-06-24

## Current Product Decision

- **Academy** is the only visible learning product in web, mobile, and admin.
- **Training** remains in the backend as a hidden, internal, legacy system for now.

## Guidance

- Do not build new frontend surfaces on Training.
- Keep Academy as the visible learning/catalog experience.
- Treat Training as a temporary backend legacy domain until a deliberate migration plan exists.
- Any future Training capability should be migrated into Academy before backend retirement.

## Notes

- This is a product/IA note only.
- No backend contract, schema, or migration changes are implied by this document.
