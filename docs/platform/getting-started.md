# Getting Started

If you are new to Fayed, start here.

This page is the shortest path to understanding the platform without having to read every document in order.

## What Fayed is

Fayed is a healthcare platform that connects patients with practitioners through a guided care journey.

The platform combines:

- discovery
- matching
- booking
- payment
- sessions
- chat
- support
- wallet and refunds
- admin operations

It is not a generic SaaS dashboard. It is a healthcare product with a calm, trust-first experience model.

## What to keep in mind

- The code is the source of truth for live behavior.
- The docs explain the product and its intended operating model.
- The design language is Clinical Warmth: calm, clear, and trustworthy.
- Money flows must stay currency-aware and policy-driven.
- Chat and cancellation states must be explained in human language, not raw technical terms.
