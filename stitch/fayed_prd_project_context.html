# Fayed - Premium Guided Care & Wellness Platform ## Product Requirements Document (PRD) ### Project Overview Fayed is a modern, guided digital care platform designed to feel like a premium wellness experience rather than a clinical medical app. It facilitates the end-to-end journey from discovery and matching to booking and session management. ### Key Design Principles * **Tone:** Calm, Supportive, Professional, Trustworthy, Simple, Elegant. * **Language:** Arabic-first (RTL), soft user-facing wording (e.g., "Your Journey", "Specialist"). Avoid "Patient". * **Visual Style:** Minimal, premium, "white-card" based, subtle depth, elegant typography, excellent spacing. ### Core User Journey 1. **Authentication:** Splash, Welcome, Login/Sign-up. 2. **Discovery:** Home Dashboard (Journey), Specialties & Sub-specialties. 3. **Matching & Selection:** Practitioner Discovery, Advanced Filters, Practitioner Profiles, Reviews. 4. **Booking & Payment:** Slot Picker, Booking Confirmation, Payment. 5. **Post-Booking:** Sessions List, Session Details, Support, Care Chat, User Profile. ### Color System (Defined) * **Primary:** #3f7dcf (with hover/active/light variants) * **Secondary:** #9eb4dd * **Accent:** #dfe7f6 * **Background:** #eff1f3 * **Surface:** #ffffff * **Text:** #1f2a3d (Primary), #4d5d79 (Secondary), #6d7c96 (Muted) * **Borders:** #d7e0ef (Light), #c1cee4 (Strong) ### Required Screens (MVP) 1.
